# Bookmarks Manager

## 概要

- raindrop.ioを模したブックマーク管理アプリを開発する

## 機能

- ブックマークの保存
- 階層構造で管理
- ブックマークの編集、移動、削除
- 非表示
- パスワード
- ログイン
- ゲスト
- サブスクリプション
- スマホ対応
- PWA

## 目的

- ペアプロやモブプロなど共同開発を通して、C#を基準に将来性のある技術をキャッチアップする
- GitHubを使った共同開発を体験する
- コーチング、ティーチング、レビュー経験を通して説明力を上げる

## 技術選定

- ASP.NET Core MVCを検討
  - 案件が多いか
  - 将来性があるか
- ランキング
  - Webフレームワークランキング
  - スタックオーバーフロー
- フリーランスエージェント
  - フリーランススタート

## 開発フロー

1. 公式ドキュメントのチュートリアルを通してフレームワーク、ライブラリの使い方を知る
1. 開発対象に置き換える
1. ローカル環境で動作確認
1. デプロイ
